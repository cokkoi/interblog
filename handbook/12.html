    排水项目--驾驶舱3D/2D地图设备打点处鼠标悬浮展示实时值

    2023/7/24 地图开发

    # 问题背景

    在排水项目临近交付的一次演示会议上，领导提出了一个问题，就是驾驶舱想要看到设备的阈值，实时值情况需要点三步，分别是点击左侧设备栏，地图出现打点，点击大殿后出现弹窗，然后在弹窗中选择第二个 tab 才会展示实时值和阈值，这明明是用户最想看到的，所以修改了设计，添加了鼠标悬浮展示相应设备的实时值,但由于驾驶舱的地图组件 3D 使用的海纳云公司自研组件，2D 使用的 DC-SDK 组件，所以需要判断当前地图是在 3D 还是 2D 环境下，分别添加该功能

    实现效果

    # 解决问题

    • 首先是 2D 地图，因为各个专项是通用的打点方法，所以单独在排水打点调用时传入参数 topic="DRAIN",即可在打点时拦截并添加相应功能，如果是排水专项，就循环打点返回的对象，给每个整个地图添加鼠标移入 mouseover 事件，再通过判断对象中的组 ID 是否是排水专项的设备打点，如果是就调用保存在浏览器中的 showDeviceValue 方法展示弹窗即可
    export const addResourcePois = async () => {
  if (topic == "DRAIN") {
    //添加防抖500ms延迟延迟
    var debounceTimer;
    points.forEach((billboard, index) => {
      billboard.on("mouseover", function (e) {
        if (billboard?.GroupID.includes("drain-dev-")) {
          clearTimeout(debounceTimer);
          debounceTimer = setTimeout(function () {
            const { UserData } = arr[index];
            window.showDeviceValue(UserData, e);
          }, 0);
        }
      });
      billboard.on("mouseout", function () {
        window.closeDeviceValue();
      });
    });
  }
};
    • 而设备的悬浮窗因为需要展示在鼠标所在的打点图标位置,所以使用 v-bind:style="{ top: position.y + 'px', left: position.x + 'px' }"方法绑定到鼠标所在位置，在 JS 中添加监控鼠标事件
    <div
  v-if="this.showDevice"

    handleMouseMove(event) {
      this.position.x = event.clientX
      this.position.y = event.clientY

      document.removeEventListener('mousemove', this.handleMouseMove)

    • 在 mounted 中将弹窗展示和隐藏方法保存到浏览器中
    window.closeDeviceValue = this.closeDeviceValue;
window.showDeviceValue = this.showDeviceValue;
    • 而在 3D 的地图下，跟2D地图基本相同，监听地图鼠标移动事件,判根据打点图标组ID判断是否鼠标在设备打点的图标上，如果是就调用handleMouseMove定位鼠标位置，再调用showDeviceValue展示弹窗内容
    //通过存在localStorage中的属性判断是否在3D地图下
let is3dMap = JSON.parse(localStorage.getItem("is3dMap")) ?? true;
if (is3dMap) {
  setTimeout(async () => {
    let lastExecutedTime = 0;
    const THROTTLE_INTERVAL = 500; // 指定的时间间隔
    await window.__g.settings.setMousePickMask(7);
    mapEvt.on("MouseMoved", (data, e) => {
      if (this.deviceMap != data.Id) {
        this.deviceMap = data.Id;
        if (data.Type == "marker" && data.GroupID?.includes("drain-dev-")) {
          const currentTime = Date.now();
          let deviceData = JSON.parse(data.UserData);
          //节流
          if (currentTime - lastExecutedTime > THROTTLE_INTERVAL) {
            document.addEventListener("mousemove", this.handleMouseMove);
            this.showDeviceValue(deviceData, e);
            lastExecutedTime = currentTime;
          }
        } else {
          this.closeDeviceValue();
        }
      }
    });
  }, 3000);
}
