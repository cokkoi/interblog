    电梯项目--websocket连接电梯实时状态与监控

    2023/7/24 websocket

    # 问题背景

    在开发青岛应急平台驾驶舱电梯项目时，需要在地图上打点所有的物联电梯，并且点击后可以查看相关信息，电梯监控和实时状态，而后端给的方案是通过 webscoket 持续心跳来保持监控地址可以播放，电梯实时状态也是通过 webscoket 每秒返回数据。

    实现效果

    # 解决问题

    • 首先在点击地图打点图标后会弹出详情弹窗，如果点击了第二个 tab，即实时监控，就会首先判断该电梯设备是否在线，离线的话展示暂无数据，因为监听了 tab 的绑定值 activeIndex，所以点击监控 tab 后会调用监控 webscoket 方法 createWebSocket(this.info[9].value)与实时状态 webscoket 方法 this.createRunSocket(this.info[11].value)
     watch: {
 activeIndex(nval, oval) {
      if (nval === 2) { //2是监控tab
        this.createWebSocket(this.info[9].value)
        this.createRunSocket(this.info[11].value)
      } else {
        this.closeWebsocket() //关闭ws服务
      }
    },
 }

    WebScoket 组件是封装好的

    • 在 createRunSocket 方法里，同样是创建一个 webscoket 对象，并将 url 地址拼接 1 传入，用 ws.onopen 方法在开启 ws 连接后直接发送电梯的设备 Code，后续就可以在 ws.onmessage 中获取该电梯的实时运行状态，并将其保存展示到页面上，同样将该 ws 对象保存到 this.runWs 中，以便后续关闭
    
 async createRunSocket(code) {
      if (!this.runWs) {
        const ws = new Websocket({
          url: realLift + '1',
        })
        ws.onopen = () => {
          if (code.length > 0) {
            ws.send(code) //发送电梯Code
          }
        }
        ws.onmessage = evt => {
          let data = evt.data
          if (!data || !(typeof data == 'string' && ~data.indexOf('{'))) {
            this.liveLiftData.runData = null
            return
          }
          try {
            data = JSON.parse(data)
            this.liveLiftData = data
          } catch (e) {
            console.log(e)
          }
        }

        this.runWs = ws
      }
    },
    • 该 ws 连接会每秒返回电梯的运行状态，所以用 watch 实时更新页面上的数据即可
        watch:{
   liveLiftData: {
      deep: true,
      immediate: true,
      async handler(data) {
        if (data) {
          this.liveLiftData = data
        } else {
          this.liveLiftData.runData = null
        }
      },
    },
    }
