    排水项目--2D地图展示排水管网

    2023/7/24 地图开发

    # 问题背景

    在公司的项目中有一项设计是要在地图上展示排水设施，其他的都还好说，只是一些点位打点，换换图标，但做着做着发现竟然有雨水管网的打点，于是便查找了下 DC-SDK 组件库的官方文档，找到了连线的方法

    实现效果

    # 解决问题

    • 首先我们清除地图上的所有图层和覆盖物的。具体来说，代码中首先判断_layers 是否存在，如果存在则遍历_layers 数组，将每个图层的覆盖物移除。然后将_layers 数组重置为空数组，以清空所有图层和覆盖物。
    this._layers &&
  this._layers.forEach((layer) => {
    layer.l.removeOverlay(layer.y);
  });
this._layers = [];
    • 通过 DC.ImageryLayerFactory.createAmapImageryLayer 方法创建了一个高德地图的图层，设置了图层的样式为 img，坐标系为 WGS84。然后通过 addBaseLayer 方法将该图层添加到地图上作为基础图层显示。

    let baselayer = DC.ImageryLayerFactory.createAmapImageryLayer({
  style: "img",
  crs: "WGS84",
});
this.viewer.addBaseLayer(baselayer);
    • 通过 For 循环使用 new DC.VectorLayer 方法创建了 VectorLayer 图层，图层的名称是一个随机生成的字符串。然后通过 this.viewer.addLayer 方法将该图层添加到地图上。

      通过 For 循环生成很多管网，并且有点击展示详情弹窗事件，必须区分，所以使用随机字符串命名

);

this.viewer.addLayer(layer);

    • 通过 new DC.Polyline 方法创建了一个 Polyline 折线对象，折线的起始点和终止点坐标分别为 list[i].startLongitude, list[i].startLatitude 和 list[i].endLongitude, list[i].endLatitude。然后通过 setStyle 方法设置了折线的样式，包括宽度、颜色等。其中，宽度设置为 2，颜色设置为蓝色。另外，还使用了 PolylineTrailMaterialProperty 材质属性，设置了折线的尾迹效果，速度为 20.0。最后，将折线对象存储到 playIon 变量中。

    playIon = new DC.Polyline(
  `${list[i].startLongitude}, ${list[i].startLatitude}; ${list[i].endLongitude}, ${list[i].endLatitude}`
).setStyle({
  width: 2,

  material: new DC.PolylineTrailMaterialProperty({
    color: DC.Color.BLUE,
    speed: 20.0,
  }),
  clampToGround: false,
});
