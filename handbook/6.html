    理解Vue2响应式原理

    2023/6/24 Vue2

    网上看到的了渡一前端公开课，觉得讲的不错，就记录一下学习笔记--Vue 响应式原理篇

    # Vue2 响应式原理

    # defineProperty 属性描述符

    想要知道 Vue2 的响应式原理，首先需要理解 defineProperty，他是属性描述符，想要获取对象的属性也很简单,就是使用 Object.getOwnPropertyDescriptor()方法

    var obj = {
  a: 1,
};
var desc = Object.getOwnPropertyDescriptor(obj, "a");
console.log(desc);

    打印的代码会有对象的值，是否可以重写，是否可以遍历等属性

    每个对象都有许多属性，属性又有相应的描述，而且描述是可以通过 Object.defineProperty 方法更改与设置

    Object.defineProperty(obj, "a", {
  value: 10, //设置属性值为10
  writable: false, //不可重写
  enumerable: false, //不可遍历
  configurable: false, //不可配置,修改
});

    除了上述属性外，属性描述符还可以配置 2 个比较重要的属性，即 get 和 set,

    Object.defineProperty(obj, "a", {
  get: function () {}, //读取器
  set: function (val) {}, //设置器
});

    一旦设置了配置了 get 属性，无论在任何地方去读取 obj.a 的时候，就会运行 get 函数

    比如 console.log(obj.a)的时候会运行 get()

    而一旦设置了配置了 set 属性，无论在任何地方去给 obj.a 赋值的时候，就会运行 set 函数

    比如 obj.a = 3+2 的时候就等于执行了 set(3+2)

    那么如何随时给属性赋值呢，就需要借助一个中间值，因为如果直接赋值的话会无限递归

    //这样写会无限递归
Object.defineProperty(obj, "a", {
  get: function () {
    return obj.a;
  },
  set: function (val) {
    obj.a = val;
  },
});

    加入中间值 internalValue

    var internalValue = undefined;
Object.defineProperty(obj, "a", {
  get: function () {
    return internalValue;
  },
  set: function (val) {
    internalValue = val;
  },
});

    # 仿写Vue2响应式

    我们首先先想一个问题，在 JS 中改变属性的值后，页面没有重新渲染，而需要在 JS 中手动操作一遍 dom 才能改变页面，比如

    
var user={
  name:'李元霸'
  age:'18'
}

// 显示姓氏
function showName() {
  document.querySelector("#firstName").textContent = "姓名：" + user.name;
}

// 显示名字
function showAge() {
  document.querySelector("#age").textContent = "年龄：" + user.age;
}

user.name = '王元霸'
showName() //改变页面展示的值

user.age = '12'
showAge() //改变页面展示的值

    这样每次改变值后都需要调用依赖该属性的函数，那如何实现自动调用呢，我们就可以利用defineProperty的get和set来实现

    • 首先我们需要观察某个对象的所有属性
    • 遍历所有属性后给每个属性添加Object.defineProperty的set和get，使用中间值实现赋值
    • 依赖收集：使用get记录是哪个函数在使用该属性
    • 派发更新：使用set来执行这个使用的函数
    • 使用一个funs数组在get中push保存这些函数
    • 在set中循环数组来执行数组中的所有函数
    • 使用ES6的new Set()函数或者if+includes保证funs数组中的方法不重复
    • 在运行每个函数前，将其保存到全局变量window.__func中，使用完后再将全局变量设为null
    • 提供一个autorun函数用来运行上一步的三个步骤
    var user = {
  name: '袁进',
  birth: '2002-5-7',
};

observe(user); // 观察

// 显示姓氏
function showFirstName() {
  document.querySelector('#firstName').textContent = '姓：' + user.name[0];
}

// 显示名字
function showLastName() {
  document.querySelector('#lastName').textContent = '名：' + user.name.slice(1);
}

// 显示年龄
function showAge() {
  var birthday = new Date(user.birth);
  var today = new Date();
  today.setHours(0), today.setMinutes(0), today.setMilliseconds(0);
  thisYearBirthday = new Date(
    today.getFullYear(),
    birthday.getMonth(),
    birthday.getDate()
  );
  var age = today.getFullYear() - birthday.getFullYear();
  if (today.getTime() < thisYearBirthday.getTime()) {
    age--;
  }
  document.querySelector('#age').textContent = '年龄：' + age;
}
autorun(showFirstName);
autorun(showLastName);
autorun(showAge);

/**
 * 观察某个对象的所有属性
 * @param {Object} obj
 */
function observe(obj) {
  for (const key in obj) {
    let internalValue = obj[key];
    let funcs = [];
    Object.defineProperty(obj, key, {
      get: function () {
        //  依赖收集，记录：是哪个函数在用我
        if (window.__func && !funcs.includes(window.__func)) {
          funcs.push(window.__func);
        }
        return internalValue;
      },
      set: function (val) {
        internalValue = val;
        // 派发更新，运行：执行用我的函数
        for (var i = 0; i < funcs.length; i++) {
          funcs[i]();
        }
      },
    });
  }
}

function autorun(fn) {
  window.__func = fn;
  fn();
  window.__func = null;
}
